# Point-Level Region Contrast for Object Detection Pre-Training

## 一句话总结

论文针对分类式自监督预训练与检测需求的错位：识别依赖对象整体语义，定位依赖点或像素级边界。作者不用单个池化向量代表区域，而在弱区域内采样多个点，让同一区域的跨视图点互为正样本，并以动量编码器产生的软亲和力蒸馏在线编码器。结果说明，点级表示不仅迁移 AP 更高，也比区域池化更能承受粗糙分区。

## 研究背景与问题

MoCo v2 把整图视为实例，容易忽略对象内部结构；区域对比虽引入局部单位，却常把前景、背景和相邻对象平均进同一向量。无标注预训练又没有真值框或掩码，因此必须回答：廉价网格能否提供有效起点，以及区域质量下降时模型是否仍保持定位能力。

## 方法总览

数据流是：图像生成两种随机增强视图及同步变换的 `4×4` 网格；基础编码器和动量编码器输出保留空间维度的特征图并上采样到 `64×64`；每个视图抽取 `N=16` 个有效区域，每区采 `P=16` 个点；区域编号组织点级对比损失 `Lc`，动量分支的点间概率分布形成亲和力蒸馏 `La`；二者构成 `Lp=αLc+(1-α)La`，再与 MoCo 图像级损失 `Lm` 合并。默认 `α=0.5`、总损失权重 `β=0.7`，教师/学生温度为 `0.07/0.1`，前 30 epoch 不启用蒸馏。

## 方法详解

设点特征为 `pi`、区域标识为 `ai`。在线视图与动量视图中，来自同一同步网格区域的点进入 InfoNCE 正集合，不同区域点和跨图像队列点进入负集合。多个点共同描述一个区域，却不经过一次平均池化，因而保留边缘与局部纹理。

亲和力蒸馏对点对相似度做 softmax。默认以动量编码器两端产生的 `A(i',k')` 为教师，以在线点和动量点产生的 `A(i,k')` 为学生，通过交叉熵对齐。网格只给训练初期的硬分组，教师提供可变化的软分组；下游迁移到 Faster R-CNN、Mask R-CNN 或 FCN 时，不保留这些预训练分支。

## 实验与证据

预训练使用 ImageNet-1K 约 128 万图像、200 epoch，或 COCO train2017 约 11.8 万图像、800 COCO epoch。ImageNet 预训练后，VOC 为 `59.4 AP/83.6 AP50/67.1 AP75`，高于 SoCo 的 `59.1 AP`；COCO 检测 `40.7 AP`、实例分割 `36.9 AP`、Cityscapes `77.0 mIoU`。COCO 预训练后，四项为 VOC `57.1`、COCO 检测 `39.8`、实例分割 `35.9`、Cityscapes `75.9`，对应 PixPro 为 `56.5/39.0/35.4/75.2`。

关键消融中，ImageNet 100 epoch 且关闭亲和力损失时，每区 4 点已达 `56.6 AP`，超过区域池化 `56.5`，64 点达 `57.6`。把 COCO 区域从真值掩码退化为真值框、`4×4`、`2×2` 网格，点级方案为 `55.5/55.2/54.9/54.5 AP`，区域级仅 `54.9/54.4/53.7/53.5`。默认蒸馏策略为 `58.0 AP`，反向教师为 `57.6`；额外动量前向也为 `58.0`，但计算更贵。VOC 中心点亲和力 Jaccard 为 `52.0`，区域级、图像级分别为 `33.8/33.1`。论文同时展示欠分割和过分割失败。

## 对 YOLO-Agent 的启发

专属 Harness 固定 YOLO 版本、骨干规模、检测微调数据、输入尺寸、增强、epoch、优化器与三个种子。**对照组**：随机初始化、MoCo v2、网格区域池化、完整点级区域对比。**指标**：`AP/AP50/AP75/APS/APM/APL`、达到目标 AP 的 epoch、预训练时长、峰值显存及小规模掩码集上的中心点亲和力 IoU。**失败判断**：三种子平均 AP 未比区域池化高 `0.5`，或 AP75/APS 下降，或移除亲和力蒸馏后收益不变，或预训练资源增加超过 25%；任一成立即不纳入默认 recipe。

## 优点

- 点级定位与区域级语义在同一目标中结合。
- 无需外部分割器，并量化了区域质量退化下的稳健性。
- 检测、实例分割、语义分割均有迁移证据。

## 局限

- 点采样、上采样和亲和力矩阵增加训练成本。
- 无监督亲和力仍可能把多个对象粘连或拆开同一对象。
- 证据基于 ResNet-50 与两阶段检测器，不能假定 YOLO 等幅受益。
