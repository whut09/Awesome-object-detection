# Angelic Patches for Improving Third-Party Object Detector Performance

## 一句话总结

该工作把攻击性贴片的优化方向反转：固定第三方检测器参数，以分类损失和边界框回归损失对局部贴片执行 reversed FGSM，并把同一类别贴片同时贴到图像内所有目标实例上，从而在腐蚀、仿射变换和未知检测器下提高目标被正确分类与定位的概率。

## 研究背景与问题

论文研究的不是“如何重新训练一个更鲁棒的检测器”，而是目标能够控制自身外观、却无法修改第三方模型的场景：例如行人穿着带特定纹理的雨衣，使自动驾驶车辆上的既有检测器更容易发现自己。模型 \(f\) 在训练和部署中均被冻结，优化变量只有类别相关贴片 \(p_i\)，因此这是面向输入侧的对象设计，而非模型侧防御。

实验对象明确覆盖 MS-COCO 中的 bus、cup、person、bottle、bowl、laptop 和 chair 七类，并以二阶段 Faster R-CNN 与一阶段 SSD 为主要模型；随后又将由二者联合训练的贴片零样本迁移到 FCOS、RetinaNet 和 YOLOv5。核心问题包括：局部覆盖能否同时改善分类与框回归、未知腐蚀下是否有效、多实例贴片能否共同优化，以及贴片能否迁移到未参与训练的检测架构。

传统鲁棒训练、数据增强或域适应需要控制模型训练过程，而且通常依赖已知分布；普通对抗贴片则通过增大损失隐藏目标。这里的目标恰好相反：在不改变检测器权重的边界条件下，设计一个面积有限、可打印、可随物体运动的“有益扰动”，把受腐蚀输入推向检测损失更低的区域。

## 方法总览

对每个类别 \(i\) 单独学习贴片 \(p_i\)。训练数据流为：读取图像 \(x\) 及该类别全部真实框 \(b\)；采样缩放变换 \(t\) 和可选腐蚀 \(\delta\)；由贴片应用算子 \(A_{\text{det}}(p,x,b,t,\delta)\) 将缩放后的贴片放入每个框中心；经过冻结检测器 \(f\)；计算检测损失 \(J_{\text{det}}\)；再只向贴片反向传播。

优化目标是

\[
p_i=\arg\min_p\ \mathbb{E}_{x,b,t,\delta}
J_{\text{det}}\!\left(A_{\text{det}}(p,x,b,t,\delta),b,i,f\right).
\]

与 FGSM 为制造错误而沿损失上升方向更新不同，reversed FGSM 使用

\[
p\leftarrow p-\epsilon\cdot
\operatorname{sign}\!\left(\nabla_pJ_{\text{det}}\right),
\]

同时降低分类头与边界框回归头的损失。

## 方法详解

### 局部覆盖与空间绑定

贴片不在背景中任意漂浮，而是根据真实框尺寸进行可微缩放并置于目标中心，由此显式绑定目标空间位置。贴片面积最多占边界框面积的四分之一，兼顾可实现性与可见特征强度。推理阶段不再需要标注、梯度或优化器，只需把已经生成并打印的类别贴片附着在真实物体上，再由任意第三方检测器正常处理图像。

### 多实例联合优化

一幅图像中同类目标可能出现多次，因此 \(A_{\text{det}}\) 会向所有对应框同时添加贴片，而不是每次仅优化一个实例。若贴片之间发生遮挡，梯度穿过完整的贴片合成过程传播，使每个副本只有实际可见区域得到更新。这一设计直接适配拥挤检测场景。

### 两种腐蚀边界

**Corruption-aware** 先贴片、再施加已知腐蚀，训练与测试使用相同腐蚀分布；论文采用 ImageNet-C 的三级 frost。**Corruption-agnostic** 则在无腐蚀图像上训练贴片，仅在测试时加入 frost、fog、brightness、contrast 等共 15 类腐蚀，检验贴片是否学到超越特定噪声模式的检测提示。

### 跨模型训练

单模型贴片从 Faster R-CNN 迁移到 SSD 或反向迁移时，只能改善部分类别。论文因此交替使用 Faster R-CNN 与 SSD 的 signed gradient 更新同一贴片，迫使纹理同时满足不同检测头和特征结构，之后直接在未参与训练的 FCOS、RetinaNet、YOLOv5 上测试。

## 实验与证据

评价协议包含 COCO 标准 AP（IoU 0.50:0.05:0.95）、AP50、AP75、不同尺寸 AP、AR，以及置信度大于 0.5、IoU 为 0.5 的高置信召回。核心控制组为 `Corrupt, NoPatch` 对 `Corrupt, Patch`，并辅以 `Clear, NoPatch` 对 `Clear, Patch`，避免把贴片收益与腐蚀本身混淆。

- 在 corruption-aware Faster R-CNN 的三级 frost 实验中，贴片将 AP 从 **14.0 提升至 22.0**，AP50 从 **24.3 提升至 39.9**，AP75 从 **22.1 提升至 34.1**；AR@100 从 **27.0 提升至 41.1**。
- 在更严格的 corruption-agnostic 对照中，训练阶段完全不见腐蚀，frost 测试 AP 仍由 **14.3 提升至 17.8**，AP50 由 **24.9 提升至 33.1**，AR@100 由 **26.6 提升至 37.9**。
- aware 与 agnostic 的受控比较表明，已知 frost 时 AP 增益为 **8.0 点**，未知 frost 时为 **3.5 点**：贴片并非只在已知腐蚀下有效，但腐蚀暴露能显著增强收益。
- 随机仿射变换施加于整幅贴片图像，即使训练时未加入对应空间增强，Faster R-CNN 与 SSD 的清晰、腐蚀场景仍获得提升。
- 物理实验覆盖 cup、person、bottle、laptop，分别在四个环境录制视频，每段约转换为 100 帧；打印贴片平均带来约 **30%** 的准确率提升，并测试了倾斜视角、遮挡、坐姿、跳跃和笔记本形变等情况。
- 双模型贴片在 FCOS、RetinaNet、YOLOv5 三个未见模型及全部目标类别上均表现出正向迁移，而单模型训练只在部分类别有效。

## 对 YOLO-Agent 的启发

### 专属 Harness

YOLO-Agent 可以把该方法实现为“固定模型、搜索对象纹理”的专用策略，而不是默认触发微调。代理应维护三个严格控制组：`无贴片+腐蚀`、`单 YOLO 模型训练贴片+腐蚀`、`YOLO 与异构检测器交替训练贴片+腐蚀`；另保留 `无贴片+清晰`，监测贴片是否损害正常域性能。

指标必须同时包含 COCO AP、AP50、AP75、AR@100，以及置信度 \(>0.5\)、IoU \(=0.5\) 的类别召回；只提高分类置信度而 AP75 不升，不能判定成功。还应分别报告中心贴片、偏移贴片、部分实例佩戴、随机仿射、15 类未见腐蚀和未见检测器结果。

一个可证伪的失败标准是：在相同测试集和随机种子下，联合训练贴片相对 `无贴片+腐蚀` 的宏平均 AP 提升不足 **3.5 点**，或 AP75 下降，或在任一未见检测器上超过半数目标类别的高置信召回降低。满足任一条件，就应拒绝“该贴片具有有效跨模型防护能力”的结论。

## 优点

- 不修改、微调或重新部署第三方检测器，部署边界清晰。
- 同时优化分类与框回归，评价也覆盖 AP75 等定位指标。
- 多实例联合贴片比单目标或背景漂浮贴片更符合检测任务。
- 包含未知腐蚀、仿射变换、跨架构迁移和物理打印验证。
- 双模型交替梯度为提升黑盒迁移提供了直接而低复杂度的方案。

## 局限

- 贴片生成依赖训练模型梯度，基础设定仍是白盒优化，黑盒能力主要来自迁移而非直接查询优化。
- 每个类别需要独立贴片，且假设目标能够改变外观并接受最多四分之一框面积的显著纹理。
- 物理测试类别、环境和每段视频规模有限，无法覆盖远距离、小目标、运动模糊与复杂天气组合。
- 与真实背景、服装材质、打印色域之间的耦合尚未被系统分解。
- 双模型训练虽改善迁移，但没有提供对任意未知检测器都成立的理论保证。

## 评分

**8.5/10。** 论文把对抗贴片的优化机制转化为对象侧鲁棒性工具，问题定义新颖，训练—推理边界明确，且具有较完整的数字、跨模型和物理证据；主要扣分点是白盒梯度依赖、物理实验规模有限，以及对更广泛黑盒模型和复杂现实分布缺少保证。
