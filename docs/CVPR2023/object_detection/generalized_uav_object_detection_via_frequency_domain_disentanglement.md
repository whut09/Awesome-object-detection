## 一句话总结

该工作面向单源域无人机目标检测：仅用白天图像训练 YOLOv5，却要求模型直接泛化到不同场景结构、夜间照明和雾天三个未知目标域。核心不是在空间特征上继续做卷积解耦，而是将源图像经 FFT 分成幅度谱 \(x_s^A\) 与相位谱 \(x_s^P\)，再用两个取值限制在 0 到 1 的可学习滤波器 \(\psi_{si}\) 和 \(\psi_{ss}\)，分别逐元素提取有利于泛化的域不变幅度谱 \(x_{si}^A\) 与不利于泛化的域特定幅度谱 \(x_{ss}^A\)。

完整数据流为：源图像 \(x_s\) 经过 FFT；幅度谱分别与 \(\psi_{si}\)、\(\psi_{ss}\) 相乘；两种幅度谱各自与原相位谱通过 IFFT 重建为 \(x_{si}\)、\(x_{ss}\)；二者共享 YOLOv5 前半段骨干 \(B_1\)，产生 \(f_{si}\)、\(f_{ss}\)；依据真实框裁剪实例特征，经 ROI-Align 统一尺寸，再由两层隐藏层 MLP 投影头 \(P\) 映射为 \(Z_i\)、\(Z_s\)，使用实例级对比损失按目标类别拉近同类、推远异类。检测路径则将域不变分量送入后续骨干 \(B_2\) 和检测头 \(H\)。

实验采用 UAVDT 与 VisDrone2019-VID，比较 Baseline、JiGen、RSC、StableNet 和 Single-DGOD，并以 AP、AP50、AP75 衡量三个未知域及其平均性能。UAVDT 单数据集设置下，本方法平均 AP50/AP75/AP 为 54.0/28.4/29.4，基线仅为 39.8/18.6/20.4；频域解耦消融也明显优于保持损失与训练策略不变的空间卷积解耦，其平均 AP 从 26.3 提升至 29.4。

## 研究背景与问题

无人机相机移动范围大，飞行高度、视角、背景结构、光照和天气会快速变化，使训练域与部署域之间出现强烈域偏移。域适应方法需要访问目标域数据，难以覆盖真正未知的现实环境；已有域泛化检测方法又多依赖普通空间卷积分离目标相关和背景相关特征，局部感受方式不足以描述无人机画面的全局外观变化。

作者先做频带拒绝实验验证前提：删除不同幅度频带会产生截然不同的泛化结果。保留全频带时平均 AP 为 20.4；拒绝 \(\alpha=0.01,\beta=0.1\) 对应频带后平均 AP 升至 24.5，而拒绝 \(\alpha=0,\beta=0.01\) 后下降至 18.1。这表明频谱并非等价贡献，可学习滤波比固定频率先验更适合检测任务。

## 方法总览

方法称为频域解耦框架。两个滤波器直接作用于幅度谱：

\[
x_{si}^{A}=x_s^{A}\otimes\psi_{si},\qquad
x_{ss}^{A}=x_s^{A}\otimes\psi_{ss}.
\]

相位谱保持共享，以尽量保留目标的空间结构；改变幅度谱则用于区分跨域稳定信息和源域外观信息。滤波器统计结果显示，\(\psi_{si}\) 在中频、高频区域的平均权重高于 \(\psi_{ss}\)，说明目标轮廓和局部结构等中高频信息更具域不变性，而低频部分更容易编码光照、天气与整体风格。

实例级对比学习不同于图像级域判别：它利用检测标注框从 \(f_{si}\)、\(f_{ss}\) 中取得每个真实目标的特征。同一分支内类别相同的实例构成正样本，其余实例作为负样本，从而要求两个滤波器分别形成内部可判别的域不变空间和域特定空间。温度参数设为 0.7，ROI-Align 后接两层隐藏层 MLP。

## 方法详解

训练采用交替优化，避免频谱解耦目标与检测目标相互冲突。参数被划分为：

- \(\theta=\{\psi_{si},\psi_{ss},P\}\)：固定检测网络，以 \(\lambda L_{con}\) 更新滤波器和投影头。
- \(\eta=\{B_1,B_2,H\}\)：固定频域模块，以 \(L_{det}=L_{reg}+L_{cls}\) 更新 YOLOv5。

最佳 \(\lambda\) 为 0.15，骨干在 Block 4 处分为 \(B_1\) 与 \(B_2\)。联合优化的平均 AP50/AP75/AP 为 53.3/27.2/28.7，交替优化达到 54.0/28.4/29.4，尤其改善严格定位指标 AP75。

## 实验与证据

UAVDT 包含约 4.1 万帧和 84 万个框。作者按原数据约定将 car、truck、bus 合并为单类；以剩余白天图像训练，以 2850 张不同场景结构的白天图像、11489 张夜间图像和 2492 张雾天图像测试。跨数据集实验则从 VisDrone2019-VID 选取 16238 张白天图像训练，再测试上述 UAVDT 三个目标域。

在 UAVDT 训练设置中，本方法十二项指标中十一项最佳，夜间域 AP 从基线 4.8 大幅提高到 20.7。以 VisDrone 训练时，平均 AP50/AP75/AP 为 47.6/21.6/24.4，相比基线 40.7/14.3/18.6 分别提升 6.9、7.3、5.8 个百分点，说明收益并非来自同一数据集划分记忆。

## 对 YOLO-Agent 的启发

对 YOLO-Agent，可将该方法实现为训练期频域泛化插件：输入端增加 FFT、双幅度滤波器和 IFFT；在骨干 Block 4 处暴露中间特征；利用标签框执行 ROI-Align 和实例投影；按批次或迭代阶段交替更新频域参数与检测参数。部署时仅保留域不变输入路径及 YOLOv5 检测网络，不需要目标域样本，也不需要在线域判别器。

若 Agent 自动选择增强策略，应把夜间和雾天性能作为主要反馈，而不能只优化源域验证集。还可记录滤波器在低、中、高频上的平均权重，监控其是否学到论文观察到的“中高频偏域不变”结构。

## 优点

主要优点是把无人机全局外观变化与频域全局建模对应起来，并将频率解耦落实到检测实例而非整幅图像。它不依赖目标域数据，兼容现有单阶段检测器；频域方案相对空间卷积方案将平均 AP50 从 50.9 提升到 54.0、AP75 从 24.6 提升到 28.4。

## 局限

局限在于实验仅以 YOLOv5 为基础检测器，类别设定与场景域仍较有限；UAVDT 中三类车辆被合并，不能证明细粒度类别下的解耦效果。雾天域 AP 为 22.3，仅略低或接近部分指标最优，说明域特定因素并未被完全去除。双路 FFT/IFFT、共享骨干前段与 ROI-Align 也会增加训练成本，而正文未给出完整效率对比。

## 评分

复现实验应设置四组严格控制：原始 YOLOv5 基线、以三层卷积块替代双滤波器的 Spatial 组、检测损失与对比损失联合更新的 Joint 组、论文完整 Frequency+Alternating 组。所有组使用相同 UAVDT 白天训练集、300 个 epoch、batch size 128、输入处理和随机种子，并分别报告三个目标域及平均 AP、AP50、AP75。

具体成功标准是完整方法至少复现平均 AP50/AP75/AP 54.0/28.4/29.4，并在平均 AP 上超过 Spatial 组约 3.1 点、超过 Joint 组约 0.7 点。若夜间 AP 未明显高于基线 4.8，或完整方法平均 AP 不高于 Spatial 组 26.3，则判定频域解耦未复现；若滤波器中高频域不变权重不再高于域特定权重，还应检查 FFT 频谱中心化、滤波器范围约束、相位共享和交替冻结逻辑。
