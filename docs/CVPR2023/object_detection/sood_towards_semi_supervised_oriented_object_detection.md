## 一句话总结

该方法建立在 Dense Pseudo-Labeling（DPL）教师—学生框架上：未标注图像分别经过教师端弱增强和学生端强增强，教师预测框经后处理确定信息区域，再从教师预测图中随机采样得到密集伪标签 \(P^t\)，同时在学生预测图的相同位置抽取 \(P^s\)，组成逐位置的伪标签—预测对。教师由学生参数的指数滑动平均更新，二者均采用旋转版 FCOS。基础无监督项包含二元交叉熵分类损失、Smooth L1 回归损失和二元交叉熵中心度损失；在此之上，Rotation-aware Adaptive Weighting（RAW）先按角度差重加权每一对样本，Global Consistency（GC）再把两组输出视为离散分布，通过最优传输建立多对多布局约束。

实验只使用 DOTA-v1.5：共 2806 张大型航拍图像、402089 个旋转实例、16 类目标，训练集、验证集和测试集分别有 1411、458、937 张图像。部分标注协议从训练集抽取 10%、20%、30% 作为有标注数据，其余作为无标注数据；完整标注协议使用全部训练集作为有标注数据、测试集作为无标注数据，统一在验证集报告 mAP。与旋转化的 Unbiased Teacher、Soft Teacher、Dense Teacher 及监督 FCOS/Faster R-CNN 比较，该方法在 10%、20%、30% 标注比例下分别达到 48.63、55.58、59.23 mAP，较同为旋转 FCOS 的 Dense Teacher 高 1.73、1.65、1.37 点；完整标注协议从 65.46 提升至 67.70 mAP，即增加 2.24 点。

关键消融明确区分了局部和全局监督的贡献。在 10% 标注设置下，纯 DPL 基线为 47.24 mAP，仅加入 RAW 为 47.82，仅加入 GC 为 47.71，二者同时启用达到 48.63；在 20% 和 30% 设置下，组合结果同样由 54.07、57.74 提升至 55.58、59.23。GC 代价矩阵只使用分数差或空间距离时分别得到 48.10 和 47.94 mAP，同时使用二者则为 48.63；RAW 的角度权重系数 \(\alpha\) 取 1、10、50、100 时依次为 47.77、47.87、48.63、47.95，说明角度信息需要被显著增强，但权重过大会放大教师错误。

## 研究背景与问题

问题的核心不是简单把水平 SSOD 的检测头替换成旋转框，而是伪标签在航拍场景中具有两类特殊不可靠性。第一，旋转角是区分密集、细长或相邻目标的重要属性，但教师角度本身可能错误，若无差别地强制学生拟合教师，会持续累积噪声。第二，飞机、车辆、船舶等对象常呈密集且规则的整体布局，逐实例的一对一损失没有利用同一图像内的位置、密度和类别置信关系。

因此，方法将监督拆成互补的两个尺度：RAW 保留对应位置上的一对一关系，根据角度分歧调节学习强度；GC 放松错误的固定对应关系，从集合层面比较教师与学生的整体布局。前者强调旋转目标最具辨识度的局部属性，后者降低单个错误伪标签对训练的支配作用。

## 方法总览

RAW 为第 \(i\) 个伪标签—预测对构造旋转调制因子：
\[
\omega_i^{rot}=1+\alpha |r_i^t-r_i^s|/\pi
\]
其中角度位于 \([-\pi/2,\pi/2)\)，论文默认 \(\alpha=50\)。最终以 \(\omega_i^{rot}\) 乘该样本的基础无监督损失。常数 1 保证角度一致时仍保留原始训练信号；角度差越大，样本对当前旋转决策边界越有信息，权重越高。其风险是教师角度严重错误时也会被放大，因此必须依赖采样控制和 GC 的集合约束共同稳定训练。

GC 首先取教师对每个采样位置预测的最高分类别 \(c(i)\)，以教师和学生在该类别上的指数化分数构造离散分布。任意教师样本 \(i\) 与学生样本 \(j\) 的传输代价由归一化二维空间距离与类别分数差之和组成，再使用 Sinkhorn 算法近似求解最优传输。这样，学生不必严格复刻每个伪标签，而是让整组预测在位置和置信结构上接近教师，实现集合到集合的多对多匹配。

## 方法详解

实现采用 ResNet-50 与 FPN，输入大图裁成 \(1024\times1024\) 图块，步长 824，相邻块重叠 200 像素。教师弱增强仅随机翻转；学生强增强包括随机翻转、颜色扰动、随机灰度和高斯模糊。模型在两张 RTX 3090 上训练 180k 次迭代，初始学习率 0.0025，于 120k 和 160k 各衰减十倍；SGD 动量为 0.9，权重衰减为 0.0001。每张 GPU 输入 3 张图像，无标注与有标注数据比例为 1:2，并通过 burn-in 初始化教师。

密集伪标签默认采样比例为 0.25。消融中比例为 0.125、0.25、0.5、1.0 时分别取得 48.27、48.63、47.91、47.69 mAP：采样过少会损失目标信息，过多则引入低质量预测和假阳性。正文有一处将最佳值写成 48.36，但表格及主结果均为 48.63，应以表格和完整实验链条为准。

## 实验与证据

贡献可以概括为三点：首次系统研究航拍场景的半监督旋转目标检测；利用角度差设计 RAW，使密集伪标签的无监督损失具备旋转感知的动态权重；利用带空间距离和分数差代价的最优传输设计 GC，把航拍布局转化为可反向传播的集合一致性约束。该设计无需改变 FCOS 的监督分支，只替换和扩展无监督训练目标，并能迁移至 CFA 与基于 RetinaNet 的 KLD，分别由 65.75、62.21 提升至 67.07、64.62 mAP。

## 对 YOLO-Agent 的启发

对 YOLO-Agent，最直接的启示是把“伪标签质量”从单一置信阈值扩展为任务属性一致性。若检测头输出旋转框，可在伪标签匹配后计算周期化角度差，并将其作为可调制的实例权重；若目标具有队列、泊位或阵列布局，则可从学生与教师候选集中抽取中心坐标和类别分数，以 Sinkhorn 距离增加集合级约束。

工程上应保留三条独立开关：DPL 基线、RAW、GC，并记录每个分支的梯度量级。GC 不应替代检测损失，而应作为较宽松的辅助约束；采样比例也不能直接设为全量，论文结果表明 0.25 明显优于 0.5 和 1.0。对不输出旋转角的普通 YOLO，RAW 不可原样使用，可改为尺度、长宽比或方向向量的一致性权重，但这属于后续扩展而非论文结论。

## 优点

局限主要来自航拍先验利用仍不充分。方法只显式建模角度与全局布局，没有处理极端尺度变化、大长宽比和小于 10 像素的实例；RAW 与 GC 还是两个分离损失，尚未形成统一的旋转—布局关系模型。GC 的多对多约束也可能在类别混杂、布局无规律或教师产生系统性漏检时对齐错误分布。

## 局限

评测 Harness 必须固定旋转 FCOS、ResNet-50-FPN、数据划分、增强策略和训练日程，设置四个核心控制组：DPL、DPL+RAW、DPL+GC、DPL+RAW+GC；主指标为 DOTA-v1.5-val mAP，分别报告 10%、20%、30% 与完整标注协议。附加控制包括采样率 0.125/0.25/0.5/1.0、GC 的距离项与分数项开关、RAW 的 \(\alpha=1/10/50/100\)，并与旋转 Dense Teacher、Soft Teacher、Unbiased Teacher 比较。

具体失败判据：若完整组合在 10% 设置下不能超过固定 DPL 基线 47.24 mAP，或不能复现 RAW+GC 高于任一单模块的互补关系，则实现失败；若 GC 同时使用距离和分数仍不高于单项代价，需检查归一化、类别索引和 Sinkhorn 梯度；若采样率 0.25 不优于全量采样，则需检查是否错误地把后处理框本身当成密集预测，而非在其信息区域中采样预测图位置。
