# Relational Matching for Weakly Semi-Supervised Oriented Object Detection

**论文**：[官方论文页面](https://openaccess.thecvf.com/content/CVPR2024/html/Wu_Relational_Matching_for_Weakly_Semi-Supervised_Oriented_Object_Detection_CVPR_2024_paper.html)

## 一句话总结

论文不把点标签直接坍缩成伪框，而是在 Mean Teacher 中围绕每个点构造旋转候选簇，对齐师生模型的候选特征关系、分类/回归排序，并用点的类别与占据关系修复双方都不确信的困难簇。

## 研究背景与问题

训练集由少量带 OBB 的图像和大量仅带类别点的图像组成。点能说明“这里有某类目标”，却不提供宽高、长宽比和角度；同一点附近可能同时存在多个合理框，逐框一致性会把增强错位和教师偏差传给学生。论文的判断是：单个候选值不稳定，但同一目标周围候选之间的相对语义与空间关系更稳定。

Rotation-Modulated Relational Graph Matching（RMRGM）以标注点为中心，按多尺度、多长宽比产生候选簇；图像和候选一起随机旋转后分别进入教师与学生。RRoI Align 提取每个旋转候选的特征作为顶点，候选两两余弦相似度作为边；顶点损失约束局部语义，边损失约束簇内结构，并用师生角度估计差调制边权，使方向分歧大的候选受到更强约束。“点中心候选图—旋转共变视图—角差加权关系边”构成其独有数据流。

Relational Rank Distribution Matching（RRDM）不硬对齐原始分数，而是分别建立簇内分类置信度和回归质量的排序分布，再把教师的相对次序传给学生。对于师生都输出低质量结果的点，困难点弱监督利用已知类别提升整簇正信号，并约束预测框对点的 occupancy，避免框有分数却不覆盖标注位置。

## 方法总览

教师由学生参数以 0.999 EMA 更新；强增强输入学生、弱增强输入教师。总损失包含全标注监督、点数据伪监督、图匹配、排序分布匹配和困难点弱监督，论文相应权重为 1.0、0.1、0.1、0.5。推理只保留教师检测器，不需要点标签与关系分支。

## 方法详解

### 1. 点中心候选图

每个点对应一个候选簇，RRoI 特征既逐顶点拉近，也转成完整相似度图。旋转增强保持候选相对图像静止，因此图边变化主要反映模型认识差异，而非几何错位。

### 2. 排序与困难簇

分类排序决定类别可信度，回归排序决定几何质量。困难簇只在两模型均困惑时由弱标签接管：类别点提供正类身份，点是否落入预测框提供最低定位约束。

### 3. 训练数据流

全标注图走 Faster R-CNN 分类与回归；点标注图同时产生教师伪框、师生候选图、两类排序分布和困难点损失。梯度只更新学生，教师随后 EMA 同步。

## 实验与证据

实验使用 DOTA-v1.0、DOTA-v1.5 与 DIOR-R，基线为 ResNet-50-FPN 的旋转 Faster R-CNN。DOTA-v1.0 仅 5% 全框标注时达到 53.85 mAP，超过 Group R-CNN 的 51.24；20% 时为 64.48 对 62.78。DOTA-v1.5 的 5% 设置为 46.29，对手为 33.99；DIOR-R 的 5% 设置为 50.76 对 48.33。全标注 DOTA-v1.0 再加入点数据时达到 74.56。

关键消融从 35.90/32.41（DOTA-v1.0/v1.5）出发：伪标注升至 43.91/38.11，图匹配到 48.78/41.89，排序匹配到 50.22/44.11，困难点弱监督最终到 53.85/46.29。只做顶点或只做边分别为 50.51/42.34、51.10/43.99，二者联合最佳；分类/回归排序和两类困难点损失也均以联合配置最好。

## 对 YOLO-Agent 的启发

Harness 固定数据划分、增强、预算和旋转 YOLO，设置“伪框基线”“+顶点蒸馏”“+关系边与角差调制”“+双排序”“+困难点 occupancy”五组。除 mAP50/mAP75，记录点覆盖率、簇内师生 Kendall 排序相关、角差分桶关系误差，并按尺度、长宽比、角度和密集度切片。若最终提升不足 0.5 mAP，或排序相关上升但点覆盖率/AP75 不升，或密集小目标下降超过 1 AP，即判定迁移失败。

## 优点

- 用候选关系与排序保存点标签的不确定性。
- 三模块分别处理关系、质量次序和共同失败，可逐项消融。
- 训练期辅助结构不增加部署推理路径。

## 局限

- 候选簇的 RRoI 两两关系增加训练显存与二次复杂度。
- 依赖教师初始质量和锚候选覆盖，细长或紧邻实例可能形成错误图。
- 点必须包含类别，对漏点、错类点与跨域噪声验证有限。
