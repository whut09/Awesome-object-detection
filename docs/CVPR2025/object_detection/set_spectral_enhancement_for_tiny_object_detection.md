# SET: Spectral Enhancement for Tiny Object Detection

**论文**: [CVF Open Access](https://openaccess.thecvf.com/content/CVPR2025/html/Sun_SET_Spectral_Enhancement_for_Tiny_Object_Detection_CVPR_2025_paper.html)  
**任务**: 微小目标检测 / 特征频谱增强

## 一句话总结

SET 在训练期把 FPN 特征分成前景与背景，只对背景执行尺度自适应的降维—升维平滑，同时向增强特征注入由检测损失梯度构造的对抗扰动，使微小目标的高频响应和梯度显著性相对杂乱背景更突出。

## 研究背景与问题

微小目标只有数个像素，经过骨干下采样后高频轮廓迅速衰减，而航拍纹理、建筑边缘和噪声仍包含大量高频成分。简单高通会同时放大背景，低通又会抹掉目标。SET 的核心判断是：利用标注把背景与目标分开处理，并用扰动敏感性迫使检测器重建弱目标表征，而不是对整张特征图施加固定频域滤波。

## 方法总览

Hierarchical Background Smoothing（HBS）依据 GT 框生成二值 mask，前景保持原样，背景进入通道压缩—扩张卷积，且卷积核随 FPN stride 增大。Adversarial Perturbation Injection（API）从分类、回归、centerness 等分支损失对每层特征求梯度，构造 L2 约束的一步对抗扰动。HBS 输出与 API 扰动相加后进入辅助检测损失，原始检测分支仍保留正常损失。

## 方法详解

在第 `i` 层，`Pi` 由 mask 拆为 `Pfg` 与 `Pbg`；`Pbg` 经过先降通道再升通道的残差映射 `φi`。信息压缩使细碎高频背景难以被完整重建，而前景像素没有经过平滑。核尺寸由 stride 决定：低层用小核清理细噪声，高层用大核处理粗粒度背景。

API 对每个检测分支构造 `ε*=ρ·∇P L/||∇P L||₂`，再按分支权重汇总为层级扰动。它不是输入攻击，而是加入 HBS 增强特征并通过辅助损失共同反传；敏感区域梯度被放大，模型被迫学习对扰动稳定的目标表示。总损失为原检测损失加各 FPN 层辅助损失，默认通道缩减率 `r=4`、扰动尺度 `ρ=1`、辅助权重 `λ=1`。

## 实验与证据

- 数据集包括 AI-TOD、VisDrone2019、DOTA-v2.0 和 COCO 2017，并按 2–8、8–16、16–32 像素报告 APvt、APt、APs。
- AI-TOD 上 FCOS 从 `12.0 AP` 提升至 `14.2`，使用 P2–P6 后从 `15.1` 提升至 `18.0`；RFLA 从 `21.7` 提升至 `24.9`，DINO-5scale 从 `23.2` 提升至 `26.6`。
- VisDrone 上 FCOS 提升 `2.0 AP`，RFLA+SET 达 `28.5 AP`；DOTA-v2.0 上 FCOS 从 `31.8` 到 `34.7`，AutoAssign 从 `33.8` 到 `35.2`。
- 组件消融：FCOS 基线 `12.0 AP`，HBS 为 `13.9`，API 为 `13.2`，两者合用 `14.2`；HBS 仅增加约 `0.9%` GFLOPs，API 主要增加训练反传成本。
- API 将示例微小目标平均显著性从 `53.99` 提至 `69.93`；频谱统计显示背景低频幅值从 `0.3787` 升至 `0.5205`，中高频被压低。

## 对 YOLO-Agent 的启发

- Harness 采用 `baseline/HBS/API/SET` 四组，固定正样本分配与 P2 使用方式；记录 APvt、APt、APs、训练秒/批、显存和推理 GFLOPs。
- 对背景纹理强度、目标尺寸和密度做切片；总体 AP 提升但 2–8 像素 APvt 不升，说明收益并非来自论文机制。
- 扫描 `r∈{2,4,8,16}`、`ρ∈{0,1,3,5}`。大扰动导致 AP 低于基线、梯度爆炸或辅助损失长期主导时判失败。
- 用 FFT 比较 HBS 前后的前景/背景频带能量；前景高频同步下降或无标注边界产生伪响应时不继续全量训练。

## 优点

- 前景保留、背景平滑与对抗增强职责明确，机制可独立否证。
- 可接入一阶段、两阶段和 Transformer 检测器，推理附加成本小。
- 同时提供 AP、显著性和频谱统计支持解释。

## 局限

- HBS 训练时依赖 GT mask，训练实现和显存管理更复杂。
- API 显著增加训练时间，且对扰动尺度敏感。
- 背景包含有效上下文时，过强平滑可能损害类别判别。
