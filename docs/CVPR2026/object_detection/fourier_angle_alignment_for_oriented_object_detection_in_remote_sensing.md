# Fourier Angle Alignment for Oriented Object Detection in Remote Sensing

**论文**：[arXiv:2602.23790](https://arxiv.org/abs/2602.23790)  
**代码**：[gcy0423/Fourier-Angle-Alignment](https://github.com/gcy0423/Fourier-Angle-Alignment)

## 一句话总结

FAA 利用“矩形长轴与其高频谱能量主方向近似垂直”的性质，从局部特征频谱估计方向；FAAFusion 用低层精确方向旋转高层语义后再融合，FAA Head 则把 RoI 对齐到 0°获得近似旋转不变特征，并与原始 RoI 残差相加供分类和旋转框回归。

## 研究背景与问题

论文把旋转检测中的误差定位到 Neck 和 Head。FPN 的低层特征保留清晰边缘与高频方向，高层特征语义强但经过多次下采样后方向粗糙；直接相加会混合不一致方向，干扰角度预测。检测头又让同一 RoI 同时服务分类和角度回归：分类希望飞机旋转后仍属于飞机，即特征旋转不变；回归则要求特征对方向变化敏感。共享特征只能折中。FAA 不再仅优化角度损失，而是在频域估计方向并显式旋转特征。

## 方法总览

Fourier Angle Alignment 包含 Fourier Angle Estimation（FAE）与 Angle Alignment。输入方形局部特征先做二维 FFT，零频移到中心，再转成极坐标能量谱；沿半径累加并按半径加权，最大角能量位置作为主方向。Angle Alignment 根据目标参考角与估计角之差旋转原特征。FAAFusion 把 FAE/FAA 放入 FPN 的低层—高层融合；FAA Head 放在 Oriented R-CNN 的 RoI Align 后，先得到规范方向特征，再与原特征残差融合，经过共享全连接层送入分类和回归分支。

## 方法详解

### 频域角度估计

二维信号旋转时，其 Fourier 频谱同角度旋转。对长宽分别为 `a>b` 的矩形，频谱是两个 sinc 项的乘积；由于长轴方向对应更窄的频谱主瓣，高频能量会沿短轴方向衰减更慢，因此主能量方向与空间长轴近似垂直。实现上，FAA 对局部特征做 FFT，乘相位因子将零频居中，转换为极坐标 `E(r,θ)`，计算 `Eθ(θ)=Σr·E(r,θ)`，并在 `[0,π)` 内取最大值得到方向。

### FAAFusion

高层特征先上采样到低层分辨率，两者经 `1×1 Conv` 压到相同中间通道，再用 unfold 取每个空间位置的局部方形块。对低层块执行 FAE 得到精细方向，以该方向为参考旋转对应高层块；所有旋转块经 fold 重建为对齐后的高层图。最终输出不是只用对齐特征，而是 `低层 + 原始上采样高层 + 对齐重建高层`，保留语义残差并补充方向一致分量。

### FAA Head

每个 RoI 特征先由 FAA 旋转到固定 0°，得到更适合分类的近似旋转不变表示。为避免旋转插值损失局部几何，规范特征与原始 RoI 残差相加，再展平通过两个共享 FC，最后分别进入分类和 OBB 回归。论文因此不是完全将分类和回归拆成两套头，而是通过“规范特征+方向敏感残差”缓和任务冲突。

FAE 输出的是局部特征主能量方向，不必等同于数据标注角。对细长矩形，两者关系较稳定；对正方形、十字形、密集相邻目标或背景纹理占主导的 RoI，频谱可能出现多个等强峰。实现应保留峰值置信度或主峰与次峰差，当方向不确定时回退原始特征，否则强制旋转会引入插值噪声。

FAAFusion 的局部块尺寸决定频率分辨率和目标纯度：块过小无法形成清晰谱峰，块过大又会混入多个目标与背景。低层方向作为高层参考的前提是两层空间位置已正确对应，因此上采样方式和 unfold/fold 的 padding 必须完全对齐。对比实验还应区分 FFT 角估计收益与“额外对齐残差分支”的容量收益，可用同结构但随机角、零角或可学习空间角作为控制。

论文在两阶段 RoI 框架中验证 FAA Head，而 YOLO 是密集点预测，缺少天然方形 RoI。迁移时必须定义局部块来自正样本中心、预测框裁剪还是动态卷积窗口，并保证训练和推理一致。若训练使用真实框角裁剪、推理改用预测角，会产生明显的教师强制落差。还要检查 FFT 对混合精度与导出的支持，避免训练可用而 TensorRT 只能回退到低效插件。

## 实验与证据

实验覆盖 DOTA-v1.0、DOTA-v1.5、HRSC2016，并把 FAA 插入 Oriented R-CNN、LSKNet-S 和 Strip R-CNN。单尺度 DOTA-v1.0 上，StripNet-S+FAA 达到 78.72 mAP，比 PKINet 高 0.33；DOTA-v1.5 上 LSKNet-S+FAA 达到 72.28 mAP，比其基线高 2.02，比 PKINet 高 0.81。HRSC2016 上，FAA 对三种基线的 COCO 风格 mAP均有提升；LSKNet 从 68.78 升至 70.74（VOC2007 设置），高 IoU 阈值 0.70—0.90 的曲线优势更明显，支持其精细角度对齐主张。

关键消融在 DOTA-v1.0、LSKNet-S 上进行：原始模型 77.49 mAP、30.98M 参数、173.68G FLOPs；仅 FAAFusion 为 77.91，仅 FAA Head 为 78.27，二者同时为 78.49、49.56M 参数、179.06G FLOPs。FAA Head 的增益更大；与 Strip Head 相比，它在 O-RCNN、LSKNet、S-RCNN 三个骨干上均取得更高 mAP，并以接近参数量显著降低 FLOPs。

## 对 YOLO-Agent 的启发

- 在 YOLO-OBB 中先试 Neck 版 FAA：只对 P2/P3 的局部块估计方向，用低层方向校正上采样高层；Head 版可把分类塔输入旋转到规范方向，而回归塔保留原特征。
- **Harness 对照组**：普通 FPN、仅空间域可学习旋转、仅 FAAFusion、分类塔规范化、完整 FAA；同时加入不使用低层参考、随机角参考和真实框角上界对照。
- **Harness 指标**：mAP50:95、AP75、平均角误差、0/90/180/270°分组 AP、高宽比区间 AP、FFT/极坐标/旋转算子的独立延迟和完整 TensorRT 延迟。
- **失败判断**：若 FAA 在近方形目标上方向估计随机且拉低 AP，需增加置信门控；若 AP50 上升但 AP75 与角误差不改善，则未复现精细对齐；若频域算子导致完整延迟超预算，不能以 FLOPs 小幅增加代替真实部署结论。

## 优点

- 从矩形频谱几何推导模块，方法与旋转检测误差具有直接联系。
- 同一 FAA 原语分别解决 Neck 跨层方向冲突与 Head 任务冲突。
- 跨三种检测器、三个数据集验证，并特别分析高 IoU 定位质量。

## 局限

- 近方形、非刚性或多主轴目标的频谱主方向可能不稳定。
- unfold、FFT、极坐标采样和旋转插值的硬件效率未由 FLOPs 充分反映。
- FAA Head 参数增长较明显，且规范化与原特征相加仍未完全解耦分类和回归。
