# Remedying Target-Domain Astigmatism for Cross-Domain Few-Shot Object Detection

**论文**: [arXiv](https://arxiv.org/abs/2603.18541)  
**代码**: 未发现论文声明的官方代码  
**任务**: 跨域少样本目标检测（CD-FSOD）

## 一句话总结

该文发现 GLIP 从 COCO 迁移到专业目标域时，Swin 层内注意力仍大范围游离，遂用 Positive Pattern Refinement 拉高同类目标响应、Negative Context Modulation 塑造统一背景边界，再以 Textual Semantic Alignment 对齐“not [class]”负概念。

## 研究背景与问题

CD-FSOD 先在大规模源域预训练，再用目标域每类 K 个标注样本微调。作者没有把问题只归结为特征分布差异，而是测量注意力距离：目标域中，前景 patch 对远处背景的权重升高，对同一物体邻域的权重下降，导致框过大、重复预测增多。普通微调虽让注意力距离略降，但目标域仍明显高于源域，论文将这种“看得到却无法聚焦”的现象命名为 target-domain Astigmatism。

## 方法总览

基线为 Swin-Tiny GLIP：图像产生多尺度视觉特征，BERT 编码类别文本，DynamicConv 融合后进入分类与定位头。微调阶段从 support boxes 内外分别统计类特定前景原型和统一背景原型，并用背景文本训练跨模态对齐；推理阶段 PPR 与 NCM 根据余弦相似度和阈值生成区域掩码，分别注入正、负原型，两个增强结果相加后送回原 GLIP 检测头。

## 方法详解

Positive Pattern Refinement（PPR）对每个类别，把 support 标注框内所有位置的视觉特征求均值得到前景原型。查询图像的每个位置与各类别原型计算余弦相似度，经温度 softmax 得到类别权重；只有最大相似度超过阈值的位置才进入前景掩码，随后把加权类原型按 γfg 注入原特征。它针对的是同物体内部注意力不足，即提高近邻前景 patch 的关联。

Negative Context Modulation（NCM）把所有真值框外区域聚合成一个背景原型，推理时用同样的相似度筛选机制调制背景位置。这里背景不分多个类别，目的不是识别场景语义，而是让与前景混淆的外围区域形成稳定参照。Textual Semantic Alignment（TSA）再用 BERT 编码“not sofa, not dog”一类由目标类别派生的负描述，把背景视觉原型和背景文本投影到共享空间，以对比式目标拉近匹配对、推远非匹配对。总损失为 GLIP 分类/定位损失加 λbg·Lctr。

从注意力距离公式看，空间位置间的距离固定，方法只能通过重新分配权重来降低前景 patch 的加权距离。PPR 提升同一对象内部近邻的 A2、A3，NCM 和 TSA 则降低远处背景的 A1；三者并不是都做“特征增强”，而是在中心与外围两侧同时改变兼容性。推理时正增强特征与负增强特征直接相加，再交给原检测头，这也意味着论文的增益来自输入特征重塑，而不是增加一套独立分类器或重新设计框回归器。

## 实验与证据

实验覆盖六个目标域：ArTaxOr、Clipart1k、DIOR、DeepFish、NEU-DET、UODD，分别包含昆虫、小样式卡通、航拍、低能见度水下、工业缺陷和复杂水下目标；指标为 COCO mAP@[0.5:0.95]，报告 1/5/10-shot。完整方法平均 mAP 分别达到 23.81、33.73、39.17；对应 GLIP 为 18.98、29.36、33.93。5-shot 中，方法在六域获得 54.98、44.83、29.41、33.87、23.64、15.66，持续高于 CD-ViTO、Domain-RAG 和 VFM-MoE 的同 backbone 重实现。

组件消融从 GLIP 的 48.11/39.28/27.39/28.40/19.55/11.40 开始，加入 NCM 后各域均增益；再加 PPR 得到 52.68/42.95/28.62/31.96/22.38/14.26；加入 TSA 后达到完整结果。作者汇总 NCM、PPR、TSA 的平均增益为 +1.06、+2.06、+1.59。注意力距离相对预训练模型的下降幅度，普通微调为 0.25%–1.30%，本文为 0.47%–1.72%。背景文本从 50 增至 200 条时 AP 从 39.34 提到 42.87，之后收益递减；NCM 阈值在 0.6–0.9 都高于基线，0.75 最佳。

六个目标数据集的规模差异很大：ArTaxOr 有 7 类、1,383 张图和 1,628 个框；Clipart1k 的测试部分为 20 类、500 张图；DIOR 为 20 类、5,000 张图和 28,810 个框；DeepFish 仅 1 类但能见度差；NEU-DET 为 6 类工业缺陷；UODD 为 3 类、506 张图和 3,218 个框。定性结果中，海事图像候选数可从 GLIP 的 100 个、普通微调的 48 个降到本文的 9 个，同时保留两个正确框。PPR 的温度与权重分析显示中等注入权重 0.1–1.0 较稳，超过 1.0 后性能明显下降，说明原型只能校正注意力，不能压过原始视觉特征。

增益在极少样本时同样明显：1-shot 平均值从 GLIP 的 18.98 提到 23.81，其中 ArTaxOr、Clipart1k、DeepFish 分别达到 27.93、38.75、30.43；10-shot 平均值为 39.17，较 GLIP 高 5.24。文本数量实验同时报告显存从 8,762MB 增到 9,245MB，200 条描述时为 9,027MB，说明作者选择 200 并非只看最高 AP，而是依据增益与内存比的拐点。

背景比例分析显示，当前景周围背景占比提高到 0.7–1.0 时，NCM 相对基线优势更明显；这与模块设计相符，因为统一背景原型只有在外围干扰充足时才提供强判别参照。论文也展示工业缺陷和水下遮挡样例，说明注意力收缩并未依赖某一种自然图像纹理。

## 对 YOLO-Agent 的启发

该思路适合做少样本域适配插件，而非永久增加主干复杂度。Harness 对照组设置为：YOLO 仅微调、仅前景类原型、前景+统一背景原型、再加入负文本对齐；同一 support seed 重复至少三次。观测指标包括六域平均 mAP、重复框数量、预测框面积偏差、前景到背景的注意力/特征距离、峰值显存和每域原型构建时间。通过阈值要求平均 mAP 提升至少 2 点、四个以上目标域为正增益、重复框数下降 15%，额外显存低于 8%。失败判断是只在 Clipart1k 单域获益，或框变少却召回下降超过 3 点，此时“聚焦”已经变成过度抑制。

## 优点

- 从层间注意力距离出发定义可观测故障，而非泛化地声称存在域偏移。
- 前景、背景和文本三种证据职责不同，组件消融能说明互补关系。
- 六个专业目标域和三种 shot 数量降低了单数据集偶然性。

## 局限

- 官方论文未给出代码链接，复现 PPR/NCM 的掩码与超参数成本较高。
- TSA 使用人工派生的简单负文本，类别规模扩大时推理和文本管理会增长。
- 所有主结果建立在 GLIP/Swin-T 上，对纯视觉 YOLO 的注意力距离定义需重新设计。
