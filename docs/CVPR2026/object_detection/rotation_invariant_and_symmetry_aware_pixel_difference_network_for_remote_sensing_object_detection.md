# Rotation Invariant and Symmetry Aware Pixel Difference Network for Remote Sensing Object Detection

**论文**：[CVF Open Access](https://openaccess.thecvf.com/content/CVPR2026/html/Zhan_Rotation_Invariant_and_Symmetry_Aware_Pixel_Difference_Network_for_Remote_CVPR_2026_paper.html)  
**代码**：[yuhua666/RIS-PiDiNet](https://github.com/yuhua666/RIS-PiDiNet)

## 一句话总结

RIS-PiDiNet 将局部像素差、Polar Harmonic Transform（PHT）与离散 SO(2) 核旋转统一成 RIS-PDC：谐波权重强调飞机、环岛等对称结构，八方向旋转卷积核的响应平均消除输入朝向偏置，从而在遮挡和任意旋转下保持稳定激活。

## 研究背景与问题

遥感检测研究常集中在旋转框参数化和小目标增强，却较少把目标自身的几何属性写入卷积。航拍目标朝向分布不均，普通 CNN 容易偏向训练中高频角度；飞机、舰船、储罐和环岛又具有双边或径向对称，局部遮挡时仍可由可见部分推断整体。标准卷积对旋转没有结构保证，大感受野只能间接学习这些规律。论文因此提出两个问题：如何在不增加参数的情况下获得稳定旋转响应，以及如何用显式对称约束补全被遮挡结构。

## 方法总览

RIS-PiDiNet 以 LSKNet 式堆叠骨干为框架，在 RIS Block、FPN 和检测头中使用 Rotation Invariant and Symmetry Aware Pixel Difference Convolution（RIS-PDC）。RIS-PDC 先从 Pixel Difference Convolution（PDC）出发，以中心或常数参考计算邻域差；Symmetry-aware PDC（S-PDC）用极坐标谐波核调制差分响应；随后将同一卷积核旋转到八个方向，对不同方向响应做平均，近似 SO(2) 群不变。网络还结合 LBP 局部描述以强化纹理，最终进入分类和回归头。

## 方法详解

### S-PDC：像素差与极坐标谐波

普通 PDC 聚合 `wi(xi-q)`，强调局部高频差异，论文固定参考值 `q=0.2` 以保持数值稳定。S-PDC 为每个邻域位置计算极坐标半径和角度，并用 `cos(2πnr²+lθ)` 形成 PHT 谐波核；多个 `(n,l)` 阶由可学习系数加权求和，再调制像素差。径向频率 `n` 与角向频率 `l` 描述不同对称模式，类内一致的对称部分被增强，非对称噪声被抑制。对于 K 重旋转或反射对称图案，作者希望变换前后的 S-PDC 响应近似相同。

### RIS-PDC：核空间旋转平均

旋转不直接作用于大尺寸特征图，而是作用于卷积核。对每个离散角度生成旋转核 `RθK`，保持输入不变并分别卷积，最后对方向响应求平均。实现采用八方向采样来近似连续 SO(2)：方向太少会欠采样，十六方向收益趋于饱和且插值成本更高。由于各方向共享原始核参数，参数量不随方向数线性增加；输出对输入旋转的变化被群平均抵消，获得近似旋转不变表示。

### 网络落点

RIS-PDC 可放在 layer1、layer2 和检测头。浅层位置主要增强局部差分与方向稳定，深层位置处理更大范围结构，检测头直接影响分类和定位。论文的 RIS-PiDiNet-S/T 将 RIS Block 堆叠进骨干，并通过 FPN输出多尺度特征；这不是单独更换损失，而是用几何卷积替换特征提取和头部的关键普通卷积。

S-PDC 与旋转平均承担不同职责：谐波核编码局部对称模式，核旋转群平均消除方向偏置。若只观察总体 AP，很难判断收益来自哪一部分；应在对称类和非对称类、均匀角度和长尾角度上分别评估。LBP 与 PDC 都强调高频差分，也可能重复放大噪声，因此噪声强度分组和背景误检率是必要补充。

核旋转的离散实现还涉及插值与网格保持。论文说明部分消融中的固定旋转采用网格保持的九十度步进，而完整配置使用八方向近似；两者不能混为同一算子。复现时要记录旋转核生成是否离线缓存、训练和推理是否共享方向集合，以及聚合使用均值还是最大值，这些细节会同时影响不变性、梯度稳定和实际延迟。

对照还应保持相同预训练权重与检测框架，避免把骨干初始化差异算作几何先验收益。

同时检查不同随机种子的稳定性。

## 实验与证据

实验覆盖 DOTA-v1.0、HRSC2016 和 DIOR-R。DOTA-v1.0 含 2,806 张图和 188,282 个实例；DIOR-R 含 23,463 张图、192,518 个旋转实例；HRSC2016 用于舰船检测。RIS-PiDiNet-S 报告 DOTA-v1.0 单尺度 78.53 mAP、多尺度 81.81，HRSC2016 98.60 mAP，DIOR-R 67.28 mAP。比较基线包括 LSKNet、PKINet、Oriented R-CNN、RoI Transformer、R3Det、S2A-Net、AO2-DETR 等；作者还用旋转输入的 ROI 响应、遮挡热图、TIDE 错误分布和 t-SNE 说明方向稳定与结构完整性。

关键消融中，LSKNet-S 式基线为 76.57 mAP、31.0M 参数、161G FLOPs。仅在检测头加入 RIS-PDC 已达 77.41；layer1+head 为 77.60、164G；三个位置全开为 77.75、206G。谐波设置表中，普通卷积 76.57，PDC 76.68，三组固定谐波与极坐标为 77.23，可学习谐波+极坐标达到最佳 77.75；去掉极坐标降到 77.31，增加到五组谐波仅 77.48，说明谐波和极坐标联合有效且存在饱和。方向消融则显示 SO(2) 八方向最好。

## 对 YOLO-Agent 的启发

- 最低风险方案是只把 YOLO-OBB 检测头第一层卷积替换为 RIS-PDC；论文中该落点成本最低且增益明显，再决定是否扩展到 P2/P3。
- **Harness 对照组**：普通卷积、PDC、PHT+Conv、SO(2)+Conv、S-PDC、SO(2)+PDC、完整 RIS-PDC；方向数比较 2/4/8/16，并加入同 FLOPs 大核卷积容量对照。
- **Harness 指标**：总体 mAP50:95、旋转角分桶 AP、遮挡比例分桶 AP、对称类/非对称类 AP、旋转前后特征一致性、TIDE 的 Loc/Miss 错误、参数、FLOPs 与真实延迟。
- **失败判断**：若收益只出现在飞机/环岛等对称类而非旋转稳定性指标，说明主要是对称增强而非群不变；若八方向核旋转在目标硬件上延迟远高于普通卷积，需采用 layer1+head 折中；若非对称类别 AP 明显下降，则谐波先验过强。

## 优点

- 用统一算子同时处理旋转不变、结构对称和局部高频，几何含义明确。
- 核旋转共享参数，不以八倍参数换取八方向响应。
- 消融覆盖模块落点、谐波阶数、极坐标、方向数和替代对称算子，证据较全面。

## 局限

- 多方向卷积使 FLOPs 从 161G 增至 206G，参数不增不代表计算免费。
- 连续 SO(2) 实际由离散方向和插值近似，严格不变性并未实现。
- 对称先验对不规则目标可能无益，固定参考值与谐波配置的跨域鲁棒性仍需验证。
