# Distilling Object Detectors with Global Knowledge

**论文**：[官方论文页面](https://www.ecva.net/papers/eccv_2022/papers_ECCV/html/2717_ECCV_2022_paper.php)  
**代码**：论文条目未提供官方代码  
**发表**：ECCV 2022

## 一句话总结

该方法从教师—学生联合特征空间中按类别选择共享原型，用实例相对这些原型的投影系数表示“全局知识”，并以师生投影差异为可靠性权重，避免把教师在模糊实例上的错误局部知识原样灌输给学生。

## 研究背景与问题

检测蒸馏常匹配特征、输出或实例关系，但这些知识大多只在教师空间内定义。目标检测包含大量背景、遮挡、小目标和误检，教师自身也会在这些位置产生噪声；若要求学生绝对复制教师的局部特征或响应，错误会被同步放大。论文观察到，实例两两关系在师生空间中的差异较大，而实例相对少量共同原型的表示更一致，因此把知识从单个实例提升为对两个特征空间整体结构的近似。

## 方法总览

框架包含 Prototype Generation Module（PGM）和 Robust Distilling Module（RDM）。PGM 在教师与学生的 TS-space 中为每个类别贪心选择 `K` 对原型，使两边都能重构实例且投影坐标尽量一致；RDM 用这些坐标构造全局蒸馏，同时用坐标差异降低不可靠实例的局部特征蒸馏和响应蒸馏权重。学生变化后，原型按周期重新生成。

这里的“全局”不是全图平均特征，而是对整个特征空间坐标系的近似。若只匹配实例两两关系，关系数量随实例数平方增长且容易受异常点影响；选出少量共同原型后，每个实例只需描述自己相对 `K` 个基向量的位置。教师和学生即使通道数不同，也能在各自空间得到同维度投影坐标，从而把异构特征比较转成坐标比较。

## 方法详解

### 1. PGM：共同基向量

教师、学生实例特征分别为 `F^t={f_i^t}`、`F^s={f_i^s}`，选出的原型是同一批实例索引对应的 `(G^t,G^s)`。目标同时最小化 `||F^t-F_I^tW^t||²`、`||F^s-F_I^sW^s||²` 与 `λ||W^s-W^t||²`，其中 `W^t、W^s` 是所有实例相对原型的坐标，`λ` 约束师生表示一致，索引约束保证每个原型在两空间中对应同一实例。论文用 matching pursuit 的贪心变体逐个选择使重构目标最小的实例，并按类别独立生成原型。

贪心选择时，`r^t_{n,i}` 与 `r^s_{n,i}` 表示前 `n` 个原型重构第 `i` 个实例后剩余的残差。候选实例依次被假设为第 `n+1` 个原型，论文根据教师残差、学生残差和新坐标一致性计算目标，选择使总目标最小者，并更新两边残差。按类别独立处理避免“person 原型去重构 car 特征”，也让原型更接近类别内有代表性的正常实例。

### 2. RDM：全局知识与可靠性

第 `i` 张图中第 `j` 个实例投影到原型后得到 `φ^t_{i,j}=P_{G^t}(f^t_{i,j})` 与 `φ^s_{i,j}=P_{G^s}(f^s_{i,j})`。全局损失为 `L_global = (1/(2NK)) Σ ρ_{i,j} ||φ^s_{i,j}-φ^t_{i,j}||²`，`N` 是实例总数，`K` 是原型数。可靠性权重 `ρ_{i,j}=1-||φ^s_{i,j}-φ^t_{i,j}||₂`：两模型对该实例的全局表示越一致，越值得蒸馏。局部特征损失把学生特征经 `1×1 conv+ReLU` 适配后与教师特征做加权平方误差；局部响应损失在 proposal 的 RoIAlign 输出上使用同样权重的 KL 散度。

总目标为 `L_kd=L_det+α1L_global+α2L_local^feat+α3L_local^resp`。FPN 的各尺度独立执行 PGM/RDM。论文默认 `K=10、λ=10`，原型每 `T=1` 个 epoch 更新；教师和当前学生都先在任务数据上预训练，再提取实例特征生成原型。

## 实验与证据

实验覆盖 PASCAL VOC 07+12 trainval→VOC07 test，以及 COCO train→val；检测器包括 Faster R-CNN 和 RetinaNet。COCO 上 Faster R-CNN 的 ResNet-101 教师为 39.8 mAP、ResNet-50 学生为 38.4；本文蒸馏后为 40.6，优于 FGFI 39.3、DeFeat 40.3、FBKD/GID 40.2。RetinaNet 的学生从 37.4 提升到 39.8，超过 38.9 的教师与 GID 39.1、DeFeat/FBKD 39.3。

VOC 消融中，Faster R-CNN 学生 AP50 为 81.3；仅局部特征、仅全局或仅响应均有增益，组合三者达到 82.9。把原型机制加入 DeFeat、RKD 和 Vanilla-KD 后，AP50 分别从 82.0→82.4、81.6→82.0、81.8→82.2。K-Means、DBSCAN 或刻意选择小/截断等模糊实例作为原型只得到 82.1–82.4、81.8 AP50，而联合 TS-space 的 PGM 为 82.9，验证“共同基”不是普通聚类中心。较大教师实验中，COCO 的 Cascade R-CNN-ResNeXt101 教师为 47.3，学生经本文方法到 41.5，与 FBKD 相当；VOC 上比 DeFeat 高 0.6 AP50。

超参数消融中，`α1=1、α2=1、α3=5、λ=10、K=10、T=1` 对应 82.9 AP50；改变各权重、原型数或更新周期时结果大多仍在 82.0 以上，但过少原型无法覆盖空间，过久不更新又会让原型滞后于正在变化的学生。检测可视化还显示，FGFI、RKD、DeFeat 会把教师的误检或不准框带给学生，而本文更常抑制这些红框错误。

## 对 YOLO-Agent 的启发

适合接在 YOLO 蒸馏管线的 FPN/PAN 多尺度输出处：按类别缓存教师与学生的正样本特征，每尺度选择 `K=10` 个共同实例原型，再同时蒸馏投影坐标和原始特征。对照组应包含普通 feature MSE、仅全局投影、仅可靠性加权局部蒸馏和完整 PGM+RDM；报告学生 COCO AP、APs、教师错误框上的蒸馏权重均值，以及原型更新耗时。若完整方案相对普通 MSE 未提升至少 0.5 AP，或原型更新使每 epoch 时间增加超过 15%，或低可靠性样本仍占总蒸馏梯度 30% 以上，应停止在线更新，改为离线原型或只保留可靠性权重。

## 优点

- 同时利用教师和学生定义知识，能显式识别教师噪声而非无条件模仿。
- 原型既构造全局知识，也为局部特征和响应提供统一可靠性尺度。
- 在一阶段、两阶段检测器以及不同教师规模上均有实验支持。

## 局限

- PGM 需要周期性提取实例特征并贪心选择原型，训练流程和存储开销高于普通蒸馏。
- `ρ=1-距离` 对特征归一化较敏感，跨实现时需确认权重范围与数值稳定性。
- 师生都需任务相关预训练；它并不是无数据或无标签蒸馏方案。

## 评分

**8.9/10**：把检测蒸馏中的教师噪声转化为师生共同空间的表示差异，概念与实验闭环完整；主要代价是原型维护使实现复杂度明显上升。
