# Out-of-Bounding-Box Triggers: A Stealthy Approach to Cheat Object Detectors

**论文**：https://www.ecva.net/papers/eccv_2024/papers_ECCV/html/8331_ECCV_2024_paper.php  
**官方代码**：https://github.com/linToTao/Out-of-bbox-attack  
**会议**：ECCV 2024

## 一句话总结

论文把通用物理触发器贴在目标框之外，通过 Feature Guidance（FG）迫使 Neck 特征接近“目标被灰块遮掉”的参考图，再用 Universal Auto-PGD（UAPGD）、EOT 与 TV loss 优化可打印纹理，使 YOLO 系列在不遮挡停车标志本体时持续漏检。

## 研究背景与问题

传统 adversarial patch 或纹理攻击覆盖目标显眼区域，虽有效却容易被发现，也可能直接破坏物体语义。框外区域对检测贡献较弱，触发器尺寸和形状还受张贴位置约束，因此从外围抑制目标更难，但更隐蔽。论文目标不是错分类，而是让指定类别的所有框置信度降到输出阈值以下。

研究以白盒通用攻击为主：同一纹理要跨图像、尺度、亮度和旋转有效，并能打印到停车标志下方；附录还考察替代模型生成触发器的黑盒迁移。

## 方法总览

给定目标框 `B`，函数 `C(B)` 选择不与目标表面重叠的外围区域，仿射变换把通用纹理 `At` 缩放、旋转、平移到该处。冻结检测器后优化三项：`Ldet` 压低目标类的框置信度乘分类置信度；`LFG` 对齐带触发图与“灰色遮掉目标”的 Neck 特征；`Ltv` 平滑像素。训练时再施加 EOT 随机噪声、对比度、亮度和旋转。

## 方法详解

FG 不破坏目标像素，而是在 Head 之前误导信息流。对原图 `x`，用灰矩形覆盖目标框得到 `xmask`，提取 Neck 层 `fz(xmask)`；触发图 `T(x,At)` 的特征被拉近该参考，使检测头看到类似“目标消失”的表示。总目标为 `Ldet+λFG LFG+λtv Ltv`。

普通 PGD 在通用、多损失优化中会震荡。UAPGD 按观察窗口记录损失最小值与方差，在进展不足或振荡时把步长减半，比固定衰减更适合通用触发器；逐渐缩小步长也使纹理更平滑。FG 并非所有超参数下都稳定，论文显示 UAPGD 与 FG 联合最可靠。

## 实验与证据

数字实验覆盖 YOLOv3、YOLOv5 等，以 Attack Success Rate 比较 PGD、UAPGD、FG 与组合；FG 会改变优化方向并加快困难样本收敛，组合通常最高。CARLA 0.9.14 使用 YOLOv3，并收集 242 张停车标志图训练触发器，记录车辆接近标志的 9 秒视频。

视频中无攻击的漏检时间占比 0%，PGD 36.67%，UAPGD 45.56%，PGD+FG 46.67%，UAPGD+FG 达 58.89%，即超过 5 秒无法识别。真实世界把打印触发器贴在标志下方：远距离时像素太小，稳定性下降；接近后攻击更稳，正面视角置信度很低，但约 ±50° 时失效。PGD 纹理由于不够平滑，实物效果弱于组合方法。

## 对 YOLO-Agent 的启发

这篇工作适合作为红队测试项。Agent 应在候选框外建立上下文敏感度图，检查外围局部纹理能否异常压低框内目标；防御不能只裁剪框内区域，因为攻击利用 backbone/neck 的大感受野。可加入随机上下文遮挡、Neck 特征一致性与时序稳定性监控。

### Harness

对照组为 clean、框内 patch、框外 PGD、框外 UAPGD、UAPGD+FG；对 YOLOv5/YOLOv8 统一触发器面积、位置和 EOT，并做打印、距离 5–30m、视角 -70° 至 70° 视频测试。观测 ASR、连续漏检时长、置信度曲线、触发器 TV、跨模型迁移率和 clean mAP。防御通过阈值：clean mAP 下降不超过 0.5 时，把组合 ASR 降低至少 50%，连续漏检不超过 1 秒；若数字有效但物理近距离仍超过 30% 漏检时长，则失败。

## 优点

- 揭示检测器对框外上下文的安全依赖。
- FG 从 Neck 特征解释外围触发器机制。
- UAPGD 针对通用多损失优化设计。
- 数字、CARLA 与真实打印形成证据链。

## 局限

- 主要为白盒攻击，黑盒稳定性有限。
- 远距离和约 ±50° 侧视角削弱攻击。
- 依赖目标框与可控张贴位置。
- 具有双重用途，应限于授权鲁棒性测试。
