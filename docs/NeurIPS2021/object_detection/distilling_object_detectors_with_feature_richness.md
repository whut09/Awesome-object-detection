# Distilling Object Detectors with Feature Richness

**论文**：[官方论文页面](https://proceedings.neurips.cc/paper_files/paper/2021/hash/29c0c0ee223856f336d7ea8052057753-Abstract.html)  
**代码**：论文条目未提供官方代码链接  
**发表**：NeurIPS 2021

## 一句话总结

Feature-Richness Score（FRS）不再用真值框硬切蒸馏区域，而是取教师每个 FPN 层所有类别分数的逐像素最大值作为软掩码，同时蒸馏 FPN 特征和分类头输出，保留框外有用响应并过滤框内教师漏检区域。

## 研究背景与问题

检测蒸馏常把真值框内区域视为前景、框外视为背景，但框并不等于“值得模仿的特征”。框外可能存在未收录类别或上下文对象，例如 mannequin 虽不属于 COCO 类别，却能提供 person 的形状知识；框内也可能有教师判断为背景的遮挡、边缘或困难区域，强迫学生匹配这些位置会传递错误。FRS 将蒸馏区域选择交给教师当前的对象响应，而不是完全依赖标注几何。

## 方法总览

教师分类头在每个 FPN 层输出类别概率图。FRS 沿类别通道取最大值，得到同分辨率、逐像素的 feature richness mask；该 mask 分别加权教师—学生 FPN 特征平方误差和分类头 BCE。不同 FPN 层使用自己的掩码，因此小目标由高分辨率层选择，大目标由低分辨率层选择。推理阶段只保留学生，FRS 没有额外开销。

FRS 的掩码是连续软值而不是 0/1 区域。对象中心通常具有完整语义与较大响应，边缘只看到局部纹理，权重自然降低；同一个物体在不同 FPN 层的响应也不同，只有与其尺度匹配的层会被突出。这样既避免把一个真值框无差别复制到所有金字塔层，也不需要额外训练 attention 网络来预测蒸馏区域。

## 方法详解

给定特征 `f`，类别概率为 `y_c=P(c|f,θ)`。论文把“该位置属于任意对象的概率”定义为 `S=max_c y_c`；实际无法枚举数据集外类别，便使用训练好的教师分类头 `θ_t` 和数据集内 `C` 类近似：`S=max_{1≤c≤C}P(c|f^t,θ_t)`。

第 `l` 个 FPN 层的掩码为 `S_l=max_c y^t_{l,c}`。FPN 蒸馏损失是

`L_FPN=Σ_l (1/N_l) Σ_{i,j} S_{l,ij} Σ_c(F^t_{l,ijc}-φ_adapt(F^s_{l,ijc}))²`，

其中 `N_l=Σ_{i,j}S_{l,ij}` 用于归一化，`φ_adapt` 是把学生通道映射到教师维度的卷积。分类头损失为 `L_head=Σ_l(1/N_l)Σ_{i,j}S_{l,ij}Σ_c BCE(y^s_{l,ijc},y^t_{l,ijc})`。总目标 `L=L_GT+αL_FPN+βL_head`，`L_GT` 是学生原始检测损失，`α、β` 平衡两种蒸馏。

这一选择与框掩码的差别可用 TP/FP/FN/TN 解释：FRS 偏向教师高响应的 TP+FP，框监督偏向真值内的 TP+FN。这里的 FP 不一定是应删除的错误，它可能是未标注但具有通用对象结构的区域；FN 则是教师在真值框内也缺乏可靠对象表征的位置。

从知识类型看，`L_FPN` 传递 backbone 与金字塔形成的中间表示，`L_head` 则约束接近最终类别决策的响应；二者共享 `S_l`，保证学生在同一位置同时学习“教师看到了什么特征”和“教师如何解释这些特征”。归一化项 `N_l` 防止某一层因高响应区域面积更大而主导总损失，特别适合各层分辨率相差较大的 FPN。

## 实验与证据

实验使用 COCO 120K train、5K val，覆盖 RetinaNet、GFL、FCOS、Faster R-CNN，训练日程为 1×12 epoch 或 2×24 epoch。RetinaNet-ResNet50 学生从 37.4 提升到 39.7 mAP，超过 ResNet101 教师的 38.9；GFL-ResNet50 在 1× 下从 40.2 到 43.6；Faster R-CNN-ResNet50 从 37.4 到 39.5；FCOS-ResNet50 从 38.5 到 40.9，也略高于 40.8 的教师。对比已有蒸馏，在 RetinaNet ResNeXt101→ResNet50 设置中本文为 40.1，超过对比方法 39.6。

提升并非只发生在宽松 IoU：RetinaNet 的 AP75 从学生 39.6 提高到 42.4，GFL 1× 从 43.3 到 47.5，Faster R-CNN 从 40.4 到 43.3。尺度指标也普遍改善，例如 GFL 1× 的 APS/APM/APL 从 23.3/44.0/52.2 升到 25.9/47.7/56.4，说明逐 FPN 层掩码确实覆盖了不同尺度，而非只增强大目标中心。

模块消融以 RetinaNet 为例：学生 37.4，单独蒸馏 FPN 为 39.4，单独分类头为 38.4，两者联合为 39.7。区域消融进一步支持 FRS：仅 TP 为 39.1，TP+FP 为 39.4，TP+FN 为 39.0，TP+TN 为 38.7；加入高丰富度的框外区域有效，而加入教师低响应的框内 FN 反而下降。论文还测得 TP 信息熵最高、FP 次之，FN/TN 最低，与“丰富度”解释一致。

论文还改变教师强度：以 RegNet RetinaNet 为学生时，教师 mAP 越高，蒸馏后的学生总体也越强，说明 FRS 并非依赖某个固定 ResNet 配对。定性结果中，蒸馏学生减少了相邻对象的重复框、背景误检和 cat/dog 混淆；这些现象与分类头蒸馏和框外高丰富度区域的共同作用相符。

## 对 YOLO-Agent 的启发

可直接在 YOLO 的 P3/P4/P5 分类 logits 上构造 `S_l=max_c sigmoid(logit^t)`，分别加权 neck feature MSE 和分类分支蒸馏；对 objectness 与 class 分离的版本，可比较只用 class、只用 objectness、二者乘积三种掩码。对照组应为全图蒸馏、GT box mask、FRS 仅 neck、FRS 仅 head、完整 FRS。报告 COCO AP、APs/APm/APl、框外高分区域贡献和教师 FN 区域梯度占比。若完整 FRS 相对 GT mask 未提升 0.4 AP，或框外高分区域中人工抽样真实对象比例低于 30%、误检类别 AP 下降超过 1 点，则提高 richness 阈值或只在 top-k 空间位置蒸馏。

## 优点

- 掩码由教师预测与 FPN 尺度共同决定，比矩形框更细粒度。
- 能利用未标注但具有对象结构的框外区域，同时规避教师框内低质量知识。
- 适配一阶段、两阶段、anchor-based 与 anchor-free 多种框架，推理零开销。

## 局限

- 最大类别分数仍受教师校准与类别偏差影响，教师自信误检会被当作丰富特征。
- 用数据集类别近似“所有对象类别”，开放世界能力只得到间接解释。
- 需要教师分类头与学生各尺度输出对齐；异构头或无 FPN 结构需重新设计映射。

## 评分

**8.9/10**：方法极简但抓住了框掩码蒸馏的根本缺陷，TP/FP/FN/TN 消融具有说服力；风险是教师高置信错误也会被强化。
