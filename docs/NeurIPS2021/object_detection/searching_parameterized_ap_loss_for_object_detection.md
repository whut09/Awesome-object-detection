# Searching Parameterized AP Loss for Object Detection

**论文**：[官方论文页面](https://proceedings.neurips.cc/paper_files/paper/2021/hash/b9009beb804fa097c04d226a8ba5102e-Abstract.html)  
**代码**：官方条目未提供代码链接  
**发表**：NeurIPS 2021

## 一句话总结

Parameterized AP Loss 把 AP 计算中负责排序、正负判定的 Heaviside 阶跃函数替换成带独立参数的单调分段线性函数，再用 PPO2 在代理训练—验证循环中搜索函数形状与定位梯度尺度，得到统一驱动分类排序和框定位的可导损失。

## 研究背景与问题

目标检测以 AP 评价分类分数排序和框定位的联合效果，但训练通常把二者拆成 Focal/Cross Entropy 与 L1/GIoU。AP 本身包含离散匹配、正负集合和排序，无法直接反向传播；已有 AP Loss、aLRP 等方法手工设计替代函数或伪梯度，仍依赖经验，而且不少方法只更新分类分支，定位分支继续使用独立回归损失，训练目标与评估指标没有真正统一。

论文认为与其手工猜一个平滑近似，不如把一族 AP 近似写进统一参数化公式，让搜索算法根据检测器实际训练后的验证 AP 选择函数。这里搜索的不是网络结构，而是 AP 中每个离散算子的可导形状；不同位置的阶跃函数使用不同参数，因为“判断正样本”“比较正负分数”“计算排名分母”等角色并不相同。

## 方法总览

方法先把 AP 显式写成分类分数 \(s_i\) 与框坐标 \(b_i\) 的函数。对预测集合 \(B=\{(b_i,s_i)\}\)，与真值匹配的为正集 \(P\)，其余为负集 \(N\)。作者定义定位分数 \(l(b_i)\)：正样本取预测框与匹配真值的 IoU，负样本取 0，以此把原本隐藏在集合划分中的定位信息放进公式。

随后把所有不可导 Heaviside \(H(\cdot)\) 替换为 \(f(\cdot;\theta)\)。每个 \(f\) 是满足端点与单调约束的分段线性函数；五处替换分别使用 \(\theta_1\ldots\theta_5\)。搜索阶段用 PPO2 采样参数，短暂训练检测器，以验证 AP 作为 reward 更新采样策略；搜索完成后固定损失，重新完整训练 RetinaNet、Faster R-CNN 或 Deformable DETR。

## 方法详解

AP 可写为正样本精度平均：

\[
AP=\frac1{|P|}\sum_{i\in P}p(i),
\]

其中 \(p(i)\) 由高于正样本 \(i\) 的正负预测数量决定，分数比较通过 \(H(s_j-s_i)\) 表示。定位分数为

\[
l(b_i)=\begin{cases}IoU(b_i,b_i^*),&i\in P,\\0,&i\in N,
\end{cases}
\]

\(b_i^*\) 是与预测 \(i\) 匹配的真值框。将集合指示和排序阶跃参数化后，论文得到统一损失

\[
L=-\frac1{|P|}\sum_{i\in B}f(l(b_i);\theta_1)
\frac{\sum_{j\in B,j\ne i}f(s_j-s_i;\theta_2)[1-f(l(b_j);\theta_3)]}
{1+\sum_{j\in B,j\ne i}f(s_j-s_i;\theta_4)f(l(b_i);\theta_5)}.
\]

\(\theta_1,\theta_5\) 控制当前预测作为正样本及其排名贡献，\(\theta_2\) 控制高分负样本错误，\(\theta_3\) 根据其他框的定位质量区分正负，\(\theta_4\) 构造排名归一化；论文阻断分母中 \(f(s_j-s_i;\theta_4)\) 的梯度以稳定训练。输入输出均归一到 \([0,1]\)。

若分段线性函数有 \(M\) 段，第 \(k\) 段为

\[
f_k(x)=\frac{y_{k+1}-y_k}{x_{k+1}-x_k}(x-x_k)+y_k,
\]

控制点满足 \((x_0,y_0)=(0,0)\)、\((x_M,y_M)=(1,1)\) 且横纵坐标单调增加。默认 \(M=5\)。论文还搜索定位分支 Gradient Scale \(\lambda\)，作用类似传统分类/回归损失权重，但服务于统一 AP 损失的反向梯度平衡。

## 实验与证据

实验使用 COCO 2017。搜索时把 118K train 划成 113K 代理训练集和 5K 代理验证集，每个候选仅训练 1 epoch；每轮采样 8 组参数，共 40 轮，PPO2 的初始函数为线性 \(f(x)=x\)。完整训练在 COCO train 上进行，val 5K 评估，基于 MMDetection 和 8 张 V100。

RetinaNet-R50-FPN 上，Focal+GIoU 为 39.2 AP，aLRP 为 39.0，Parameterized AP Loss 为 40.5；Faster R-CNN-R50-FPN 中 Cross Entropy+GIoU 为 39.1、CSE-AutoLoss-A 为 40.4、本文为 42.0；Deformable DETR-R50 从原始 Focal+L1+GIoU 的 43.8 提升到 45.3。论文总结相对手工损失可提升 1.5–3.0 AP。

消融说明搜索不可替代：Sigmoid、sqrt、linear、square 替代分别仅 3.2、2.3、22.9、36.4 AP，搜索函数为 40.5。五处共享参数为 37.8，独立参数为 40.5；不使用搜索到的梯度尺度为 39.4，使用后为 40.5。定位度量用 L1、IoU、GIoU 时分别为 40.0、38.1、40.5，IoU 因无重叠时梯度弱而落后。分段数 3/5/7 为 34.2/40.5/40.3，5 段已足够。PPO2 在四次重复中比随机搜索更快找到高 reward 参数。

RetinaNet 的手工替代结果跨度极大，也说明“可导”本身远远不够：Sigmoid 和平方根近似甚至难以形成有效训练，线性初始化只有 22.9 AP，平方函数虽然达到 36.4，仍低于常规检测损失。搜索并非只在这些固定函数中选择，而是移动五段折线的内部控制点，使不同阶跃项分别获得适合正负判定、排序和归一化的斜率。

搜索阶段与正式训练明确分离。代理阶段只用一轮网络训练来评估一组损失参数，PPO2 根据验证 AP 更新截断正态采样分布；最终报告结果则固定搜索到的函数，在完整 COCO train 上从头训练。这个流程避免把搜索过程中已见过验证集的网络权重直接用于最终模型，但损失形状仍可能对代理划分过拟合，因此迁移到新数据时应重新验证。

## 对 YOLO-Agent 的启发

接入点不是简单替换某一条 BCE，而是需要在 YOLO 分配完成后构造当前 mini-batch、逐类别的预测集合，保留分类分数、解码框及匹配真值，计算参数化 AP 目标并同时回传分类与定位。首轮应把 YOLO 原 BCE/DFL/IoU 损失作为基线，另设手工 square 近似、共享参数搜索、独立参数搜索、固定/搜索梯度尺度四组；搜索阶段使用短程代理训练，最终必须从相同初始化完整重训。

对照指标应包括 AP、AP75、APS、搜索成本和训练稳定性。论文中独立参数相对共享参数高 2.7 AP、搜索梯度尺度高 1.1 AP，因此失败阈值可设为：独立搜索相对原 YOLO 总 AP 增益低于 1.0，或 AP75 无正增益；若短程 reward 提升但完整训练 AP 下降超过 0.5，说明代理任务与正式训练不一致。还应设置数值失败阈值：任一轮出现 loss 非有限值或正样本为空时跳过更新，因为该损失依赖 batch 内全局排序，远比逐点 BCE 敏感。

## 优点

- 从 AP 原公式出发统一分类排序与定位，不再依赖两条独立损失的人为权重组合。
- 搜索空间有端点、单调和分段结构约束，比任意符号搜索更可控。
- 在 one-stage、two-stage 和 transformer detector 上均优于对应常用损失。

## 局限

- 双层搜索需要反复短训检测器，成本明显高于手工损失调参。
- 损失涉及 batch 内预测两两比较，计算和显存随候选数增长，密集 YOLO 必须先合理筛选。
- 搜索结果可能依赖数据划分、检测器和训练日程，迁移到新模型未必仍最优。

## 评分

- **创新性：9/10**——把不可导 AP 近似转化为结构化损失函数搜索。
- **实验充分性：8.5/10**——覆盖三类检测器与关键搜索设计，但搜索成本分析有限。
- **可复现性：7.5/10**——公式和搜索配置完整，计算资源与双层流程门槛较高。
- **对 YOLO-Agent 价值：8/10**——适合自动损失研究，但需处理密集候选的二次复杂度。
