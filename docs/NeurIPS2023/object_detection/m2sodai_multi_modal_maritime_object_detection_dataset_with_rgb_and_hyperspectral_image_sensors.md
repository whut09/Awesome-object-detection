## 一句话总结

研究团队使用 Cessna Grand Caravan 208B 单引擎飞机，在机腹同时安装向下拍摄的 DMC RGB 相机与 AisaFENIX 高光谱传感器，以约 1 km 高度、260 km/h 速度完成 12 次测量任务、59 条航带，覆盖韩国沿海 299.7 km²。原始 RGB 与 HSI 虽同时采集，却存在空间偏移，因此先转灰度并用 CLAHE 增强，再分别提取 ORB 特征，经暴力匹配器寻找对应点，以最小二乘估计单应矩阵，完成配准后裁成 `1600×1600×3` RGB 与 `224×224×127` HSI 对。最终得到 1,257 对同步影像、11,527 个实例，按 1,007/125/125 划分训练、验证和测试集。

DoubleFPN 不是把两种图像直接拼接，而是采用双主干—双颈部的中层融合数据流。RGB 与经增量 PCA 从 127 波段压缩至 30 个主成分的 HSI 分别进入 ResNet-50 主干；30 个主成分保留超过 99.9% 的累计方差。HSI 分支各尺度特征 \(C_{HSI,i}\) 经单通道 `1×1` 卷积生成注意力，最高层接 Sigmoid，随后在自顶向下路径中与上一层二倍上采样注意力相加，形成 \(H_i\)。每个 \(H_i\) 再上采样到对应 RGB 特征尺度，与 \(C_{RGB,i}\) 逐元素相乘；所得 \(H_i\cdot C_{RGB,i}\) 进入标准 FPN 和 `3×3` 卷积，最后由 Faster R-CNN 预测类别及边界框。

在统一 ResNet-50、Faster R-CNN 和从零训练条件下，DoubleFPN 达到 44.4% mAP、84.8% AP@.5、39.3% AP@.75；RGB-only FPN 为 38.8% mAP，而 HSI-only 仅为 7.8%。这一模态消融说明：加入 HSI 带来 **5.6 个 mAP 点**，但用低空间分辨率 HSI 替代 RGB 会损失 31.0 点。DoubleFPN 也高于 UA-CMDet 的 42.9%、DetFusion 的 42.0% 和 Early Fusion 的 42.9%。在含海面效应的子集上，其 mAP 为 42.2%，较全测试集只下降 2.2 点；RGB-only 降至 35.1%，下降 4.4 点。

## 研究背景与问题

核心问题不是普通船舶检测，而是阳光反射、波浪和风造成的海面纹理会在 RGB 中呈现出与漂浮物相似的局部外观，引发大量假阳性。高光谱数据覆盖 400–1000 nm、以约 4.5 nm 间隔采集 127 个波段，可从少量像素揭示材料反射特性。论文的光谱分析显示，漂浮物与海面效应在 RGB 裁块中可能相似，但其高光谱反射强度曲线不同，因此 HSI 更适合承担“材料判别与抑制误检”的辅助角色。

RGB 空间分辨率约 0.1 m，HSI 约 0.7 m，两者存在约七倍尺度差异。DoubleFPN 据此没有要求 HSI 独立恢复精确轮廓，而是把它转成跨尺度注意力，调制高分辨率 RGB 特征。这一设计同时规避了早期融合难以建立异构原始通道共同表示，以及晚期融合无法充分学习特征相关性的问题。

## 方法总览

数据集包含“船舶”和“漂浮物”两类，每类约 5,764 个实例。漂浮物包括浮标、救生圈、小型救生艇、冲浪板以及穿救生衣的人体模型；船舶包括汽船、游轮、渔船、帆船、筏及其他船型。两名作者使用 Labelme 标注最小包围框，并进行多轮交叉检查；涉及救援情境时使用人体模型而非让真人处于危险中。

与既有航空 RGB 数据集或像素级 HSI 遥感数据不同，该数据集同时具备航空视角、同步 RGB/HSI、目标级边界框三项属性。其价值不仅是规模，更在于提供了可直接研究“高空间分辨率外观＋低空间分辨率光谱”协同检测的受控基准。

## 方法详解

训练使用两张 NVIDIA RTX 3090，共 73 个 epoch；每卡 batch size 为 1，SGD 学习率 `2×10⁻²`、动量 0.9、权重衰减 `1×10⁻⁴`。DoubleFPN 的 RGB 与 HSI 主干分别输出五层和四层特征，并对齐为 \(N=5\)。评估采用 COCO 指标：IoU 0.5–0.95 平均的 mAP、AP@.5、AP@.75，以及 APs、APm、APl和两个类别的 AP。

DoubleFPN 的船舶 AP 为 55.7%，漂浮物 AP 为 33.1%；RGB-only 分别为 52.4% 和 25.2%，说明主要增益集中于最容易和海面效应混淆的漂浮物，提升 7.9 点。小目标 AP 从 RGB-only 的 18.3% 提升到 35.2%，也符合光谱信息能从有限像素提供材料线索的动机。

## 对 YOLO-Agent 的启发

复现 Harness 应设置四个严格控制组：`RGB-only YOLO + FPN/PAN`、`HSI-only YOLO`、`RGB+HSI 通道级早期融合 YOLO`、`RGB+HSI DoubleFPN-style 注意力融合 YOLO`。所有组必须使用相同数据划分、输入裁块、增强策略、优化器、训练轮数、随机种子集合和检测头容量；HSI 统一采用同一增量 PCA 模型压缩为 30 维，PCA 只能在训练集拟合，防止测试信息泄漏。

主指标为 mAP、AP@.5、AP@.75、船舶 AP、漂浮物 AP、APs，并单独报告含海面效应子集的 mAP 及其相对完整测试集的下降量。附加效率指标应包括参数量、FLOPs、端到端延迟和显存占用，以验证高光谱分支的工程代价。

具体失败判据：若多模态模型在至少三个固定种子上的平均 mAP 未超过 RGB-only **3.0 点**，或漂浮物 AP 未提升 **5.0 点**，则不能认为光谱融合有效；若海面效应子集上的 mAP 降幅不小于 RGB-only，说明模型没有获得论文声称的抗海面干扰能力；若 HSI-only 异常接近 RGB-only，还应优先检查配准泄漏、分辨率处理或测试集 PCA 拟合错误。

## 优点

最重要的消融是模态角色而非网络深度：RGB-only 38.8%、HSI-only 7.8%、两者融合 44.4%。HSI 单独检测能力弱，却能通过注意力显著增强 RGB，证明其贡献是“抑制错误响应”，而不是提供精细几何。Early Fusion 与 UA-CMDet 同为 42.9%，仍低于 DoubleFPN 1.5 点；DetFusion 低 2.4 点，支持中层、逐尺度联合融合的必要性。

## 局限

局限首先来自地域与采集成本：数据全部采自韩国沿海，船型、目标分布和海况可能具有区域偏差；固定翼飞机与专业 HSI 设备昂贵，也限制扩展速度。其次，HSI 的低空间分辨率使其无法替代 RGB，且额外光谱分支显著增加推理延迟。作者因此建议探索更轻量的 3D CNN 高光谱特征映射。

此外，随机影像划分可能无法完全检验跨航带、跨日期或跨海域泛化。更严格的后续基准应按飞行任务或地理区域隔离训练与测试，并分别统计不同反光强度、浪况、目标距离和目标尺度下的性能。

## 评分

这项工作的关键启示是：异构传感器不必在每项能力上都更强，只要其弱分辨率数据能提供正交信息，就可以作为高分辨率视觉特征的门控信号。对海事小目标而言，RGB 决定“在哪里、形状如何”，HSI 帮助判断“该区域是什么材料”，DoubleFPN 将这种职责分工直接编码进特征金字塔。

对 YOLO 系统的合理迁移不是简单把 30 个主成分与 RGB 拼成 33 通道，而是在多尺度颈部生成 HSI 注意力，并逐级调制 RGB 的 P3–P7 特征。真正值得验证的目标也不是完整测试集上微小的总体提升，而是漂浮物、小目标及强反光海面中的假阳性是否稳定下降。
